# Portfolio Website

A simple portfolio website built with HTML, CSS, and JavaScript.

## Files

- `index.html`: The main HTML structure
- `styles.css`: CSS styles for the website
- `script.js`: JavaScript for interactivity

## How to View

Open `index.html` in your web browser to view the portfolio.

## Customization

Edit the content in `index.html` to personalize your portfolio. Update `styles.css` for styling changes and `script.js` for additional functionality.
